# 🌸 粉色主题硬件监控网站

一个可爱的粉色主题硬件监控网站，具有星星墙分布效果，能够显示本地硬件数据和温度信息。

![技术栈](https://img.shields.io/badge/Tech-HTML5%20%7C%20CSS3%20%7C%20JavaScript-blue)
![主题](https://img.shields.io/badge/Theme-Pink%20Cute-ff69b4)

## ✨ 特性

- 🎨 **可爱粉色主题** - 温馨的粉色配色方案
- ⭐ **星星墙布局** - 硬件信息卡片如星星般随机分布
- 🌡️ **实时温度监控** - 智能温度预警系统
- 💫 **流畅动画效果** - 卡片浮动、悬停、闪烁动画
- 📱 **响应式设计** - 完美适配手机、平板、桌面
- 🎯 **温度颜色预警** - 直观的颜色渐变指示系统

## 🎮 功能说明

### 硬件监控组件

#### 🖥️ CPU 监控
- 处理器型号和核心数
- 实时使用率百分比
- 温度监控和可视化
- 运行频率显示

#### 🎮 GPU 监控  
- 显卡型号信息
- GPU 使用率
- 温度监控
- 显存使用情况

#### 🧠 内存监控
- 内存型号和总容量
- 实时使用量统计
- 内存使用率百分比
- 内存频率信息

#### 💾 存储监控
- 硬盘型号和总容量
- 存储空间使用情况
- 硬盘温度监控
- 健康度状态指示

#### 🔧 系统监控
- 主板型号和温度
- 系统电压监控
- 风扇转速和状态
- 系统整体健康度

### 视觉效果系统

#### 🌡️ 温度颜色指示
- 🟢 **绿色** (< 50°C): 正常运行
- 🟡 **黄色** (50-70°C): 需要注意
- 🟠 **橙色** (70-85°C): 警告状态
- 🔴 **红色** (> 85°C): 危险高温

#### ✨ 动画效果
- **卡片浮动**: 轻微上下浮动效果
- **悬停放大**: 鼠标悬停时的缩放效果
- **温度条动画**: 动态填充和闪光效果
- **星星背景**: 背景星星的闪烁动画
- **高温警告**: 高温时的脉冲边框效果

## 🔧 自定义配置

### 更换背景图片
1. 将您的背景图片放入 `assets` 文件夹
2. 在 `css/style.css` 中修改背景图片路径：

```css
body {
    background-image: url('../assets/your-background.png');
}
```

### 调整更新频率
在 `js/main.js` 中修改更新间隔：

```javascript
this.UPDATE_INTERVAL = 2000; // 毫秒，默认2秒
```

### 自定义颜色主题
在 `css/style.css` 的 `:root` 部分修改 CSS 变量：

```css
:root {
    --primary-pink: #ff9fdc;     /* 主要粉色 */
    --light-pink: #ffb3e6;       /* 浅粉色 */
    --dark-pink: #e680c2;        /* 深粉色 */
    --accent-purple: #d4a4ff;    /* 紫色点缀 */
}
```

### 调整卡片分布
修改 `js/main.js` 中的分布参数：

```javascript
const cardWidth = 300;    // 卡片宽度
const cardHeight = 200;   // 卡片高度
const margin = 20;        // 卡片间距
```

## 🛠️ 开发说明

### 技术栈
- **HTML5**: 语义化标签，现代标准
- **CSS3**: Grid布局，Flexbox，CSS变量，高级动画
- **JavaScript ES6+**: 类、模块、箭头函数，现代语法

### 核心算法
1. **星星分布算法**: 智能防重叠的随机位置生成
2. **数据模拟算法**: 多波形叠加的真实数据波动
3. **温度映射算法**: 温度值到颜色的平滑渐变
4. **响应式算法**: 设备尺寸自适应布局

### 性能优化

- 内存泄漏防护
- 页面可见性 API
